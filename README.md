# Kabinki.kz — Калькулятор душевых кабин

## Описание
Мобильное веб-приложение для расчёта стоимости душевых кабин и генерации коммерческого предложения в PDF.

## Файлы
- index.html — Калькулятор (страница 1)

## Цвета стекла
- Прозрачная
- Матовая
- Бронза
- Графит

## Цвета фурнитуры
- Хром
- Чёрно-матовый
- Золото
- Латунь

## Использование
1. Откройте index.html на телефоне (Android/iOS)
2. Введите размеры стёкол
3. Выберите параметры
4. Нажмите "Создать КП (PDF)"
5. PDF скачается автоматически
